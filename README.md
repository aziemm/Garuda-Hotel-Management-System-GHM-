# 🏨 Garuda Hotel Management System (GHM)

A web-based Hotel Management System developed to simplify hotel booking and management processes for both customers and administrators.

---

## 📖 Overview

Garuda Hotel Management System (GHM) is a PHP-based web application that allows customers to book hotel rooms online while providing administrators with tools to manage bookings, guests, and staff efficiently.

The system provides separate interfaces for users and administrators to ensure secure and organized management of hotel operations.

---

## ✨ Features

### 👤 User Module

- User Login
- Forgot Password
- Hotel Room Booking
- Multi-Step Booking Process
- Booking Confirmation
- Booking History
- Cancel Booking
- Guest Profile Management

### 👨‍💼 Admin Module

- Secure Admin Login
- Dashboard
- Staff Management
- Add Staff
- Edit Staff
- Delete Staff
- Guest Management
- Booking Management
- View Booking Information

## 🚀 Installation

1. Clone this repository

```bash
git clone https://github.com/yourusername/Garuda-Hotel-Management-System-GHM.git
```

2. Move the project into your web server directory.

Example:

```
xampp/htdocs/GHM
```

3. Import the database into your database server.

4. Configure the database connection.

5. Start Apache and Database services.

6. Open your browser

```
http://localhost/GHM
```
